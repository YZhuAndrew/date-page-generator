# 日期页面生成器

这是一个简单的网页应用，可以生成带有当前日期的页面。应用包含一个主页，提供按钮用于生成新页面，以及一个侧边栏用于浏览已生成的页面。

## 功能特点

- 点击按钮生成新的日期页面
- 生成的页面包含问候语和当前日期时间
- 主页侧边栏显示已生成页面列表
- 页面具有现代化的UI设计和动画效果
- 使用 localStorage 保存已生成页面的信息

## 技术栈

- HTML5
- CSS3 (含动画和渐变效果)
- JavaScript (含 localStorage 操作)
- Vercel 部署配置

## 部署方式

本项目可以通过 Vercel 直接部署。只需将代码推送到 GitHub 仓库，然后在 Vercel 中连接该仓库即可自动部署。

### 本地运行

1. 克隆仓库到本地
2. 使用浏览器打开 `index.html` 文件即可

### Vercel 部署

1. 将代码推送到 GitHub 仓库
2. 在 Vercel 中导入项目
3. 选择对应的 GitHub 仓库
4. Vercel 会自动检测并使用 vercel.json 配置文件进行部署

## 使用说明

1. 打开主页
2. 点击"生成新页面"按钮
3. 新页面会自动生成并下载
4. 在侧边栏可以查看已生成的页面列表
5. 点击列表中的页面链接可以查看对应页面

## 注意事项

由于浏览器安全限制，本地运行时生成的页面会作为文件下载而不是直接保存在项目目录中。在服务器环境中运行时，可以通过后端代码实现文件的直接保存。

当前版本使用 localStorage 保存已生成页面的信息，这意味着数据只在当前浏览器中可用。在实际应用中，应该使用后端数据库来存储这些信息。
